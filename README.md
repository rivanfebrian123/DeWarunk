### Cara menjalankan
Ketik dan enter perintah2 ini di konsol kamu:

```
npm install
npm start
```

### Peringatan untuk pengguna Windows
Gunakan Windows Terminal untuk menjalankan aplikasi ini, jangan
pake konsol biasa, supaya tampilan optimal
